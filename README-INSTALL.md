# Neostrap UI

Neo-brutalist React component library built on shadcn/ui.

## Installation

Add the button component to your project:

```bash
npx @neostrap/ui add button
```

## Usage

```tsx
import { Button } from "@/components/ui/button"

export default function App() {
  return (
    <div>
      <Button>Brutal Button</Button>
      <Button variant="outline">Outline</Button>
      <Button variant="ghost">Ghost</Button>
    </div>
  )
}
```

## Features

- **Neo-brutalist design** - Bold, high-contrast, geometric buttons
- **Multiple variants** - brutal, default, outline, secondary, ghost, link, destructive
- **TypeScript support** - Fully typed components
- **Tailwind CSS** - Built with Tailwind CSS v4
- **Easy to customize** - Extend and override styles

## Components

- `Button` - Neo-brutalist button component with multiple variants
